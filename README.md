# QiitaArticleSearchApp

52歳未経験男性が0からカリキュラム内で作成したオリジナルアプリ

# DEMO
準備です

# Features
QiitaのJSONデータ取得
パースして一覧表示
Webページ画面遷移
検索機能
検索記事一覧画面遷移
検索記事Webページ画面遷移
検索記事お気に入り保存機能
お気に入り記事一覧から削除編集機能
Firebaseサービスを利用したリモートプッシュ通知機能

AppStoreへリリースへ向け奮闘中。

# Requirement

platform :ios9.0　以上
'Alamofire', '~> 4.9.1'
